Sayang Smash — Web App Skeleton (Next.js + tRPC + Drizzle)

This repository hosts the Sayang Smash project. The actual application lives in the subfolder `sayang_smash/app` using Next.js with the Pages Router, tRPC for the API layer, and Drizzle ORM for DB access to a Supabase Postgres database.

How to run (in this repo):
- Ensure Node.js is installed.
- cd sayang_smash/app
- npm install
- npm run dev

This is a scaffold. Secrets must live in .env (not committed). See .env.example in the subfolder.
